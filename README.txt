# 情侣交往天数计数器

这是一个简约风格的情侣交往天数计数器网页，可以实时显示你们在一起的时间。

## 功能特点

- 简约MUJI风格设计
- 实时计算并显示交往天数、小时、分钟和秒数
- 可以上传和显示两人的头像
- 可以添加自定义留言
- 自定义你们的纪念日
- 数据保存在本地，无需登录

## 如何使用

1. 访问网页：https://你的用户名.github.io/love-counter/
2. 设置你们的纪念日
3. 上传你们的头像（点击头像上的相机图标）
4. 输入你们的名字
5. 添加你们的爱情故事

## 如何部署到GitHub Pages

1. 创建一个名为 `love-counter` 的GitHub仓库
2. 将所有文件上传到仓库：
   - index.html
   - style.css
   - script.js
   - README.md
3. 在仓库设置中启用GitHub Pages，选择main分支作为源

## 技术说明

- 使用HTML5、CSS3和JavaScript构建
- 数据存储使用localStorage，保证信息保存在本地设备
- 响应式设计，适配手机和电脑
- 不需要任何后端服务器

## 自定义

你可以通过修改 `style.css` 文件来自定义网页的颜色和样式。
